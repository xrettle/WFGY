## What Counts As A Real Objection

| Objection type | Valid target |
|---|---|
| theorem mismatch | The formal target does not match the original problem. |
| proof gap | A required proof step is missing. |
| fake formalization | Status labels or metadata are used instead of proof. |
| weak route | A generated route does not connect to the target. |
| hidden assumption | A key assumption is not stated. |
| failed regeneration | Re-shooting does not recover the claimed route skeleton. |

---

## What Does Not Count

| Weak objection | Why |
|---|---|
| “I do not believe it.” | Belief is not review. Inspect the files. |
| “This is too big.” | Size is not an objection. Find the break point. |
| “AI generated it.” | Attack the route, proof terms, and verification materials. |
| “I dislike the style.” | Style is not the mathematical target. |
