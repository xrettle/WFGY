# WFGY 5.0 Avatar Archive

# WFGY 5.0 Polaris Protocol

The active WFGY 5.0 entry point is now here:

[Go to WFGY 5.0 Polaris Protocol](https://github.com/onestardao/WFGY/blob/main/Polaris/README.md)

This Avatar folder is now preserved as a legacy archive.

It records an earlier public direction of WFGY 5.0, when the system was still presented as an Avatar runtime for engineered language behavior, route tuning, reusable builds, and persona recovery operations.

The project has since moved forward.

The new official name is WFGY 5.0 Polaris Protocol.

## Why this folder is archived

This page is kept for historical continuity.

Some readers may have discovered WFGY through Avatar.
Some may have tested the early runtime.
Some may still want to study how the original WFGY 5.0 direction was shaped before the Polaris upgrade.

For that reason, this folder is not deleted.

It is archived as a collectible legacy layer.
